The backend server needs to be started before you can use the application. This is normal for the first time setup!

Quick Start

  1. Start the Backend

    Open a terminal and run:

    chmod +x start-backend.sh
    ./start-backend.sh

    Wait for: "✅ Backend Started Successfully!"

  2. Start the Frontend

    Open a new terminal and run:

    npm run dev

  3. Login

    Open http://localhost:5173 and wait for the green checkmark, then click "Demo Login as Traveller"
